Mes Projets

Projet 1

Refonte du site de l'IUT

Durant le semestre 1 de mon DUT informatique, nous avons réalisé par groupe de 3 étudiants la restauration du site de l'IUT de Lens. L'objectif était de réaliser un site web servant à guider les étudiants et les entreprises à en savoir plus sur l'IUT. Le site devait être autant esthétique qu'informatif. Nous nous sommes donc réparti les tâches au sein du groupe afin d'aboutir au résultat final.

Projet 2

Création du labyrinthe de ravensburger

Pour le 2ème semestre, nous avons réalisé le labyrinthe de ravensburger dans le langage Java. L'objectif était de créer des joueurs humains et des joueurs ordinateurs afin de pouvoir jouer par 2 ou 3 sur le plateau de jeu. Nous étions 4 pour ce projet et nous avons décidé de faire ce jeu dans le thème "Naruto".

Projet 3

Création d'un sokoban

En fin de semestre 2, nous avons réalisé par groupe 4 un sokoban dans le langage Python et de la librairie PyQt4. L'objectif était de réinventer le jeu sokoban en y proposant une variante. Pour cela, nous avons fait ce jeu avec deux thèmes : "Naruto" et "Mario Remake".

Projet 4

Création d'une base de données

Durant le semestre 3 de ma 2ème année de DUT, nous avons créé par binôme une base de données qui concernait le domaine vétérinaire. C'est l'un des projets que j'ai le plus aimé car le choix du sujet était libre.

Projet 5

Compteur de mots

Ce projet est en cours de conception. Il consiste à réaliser par binôme un compteur de mots dans le langage java à l'aide de l'outil Scanner. De plus, ce compteur doit être créé avec différentes méthodes telles qu'un tableau, une liste chaînée, un arbre binaire et une table de hashages.

Projet 6

Marathon du web

Le "Marathon du web" est un projet phare de l'IUT de Lens qui rassemble 3 étudiants en informatique et 4 étudiants en MMI pour clôturer le semestre 3. Dans ce projet, nous avions à réaliser un site web en 35 heures qui permettaient de référencer des séries. Pour cela, nous avons utilisé PHP et son framework Laravel ainsi que MySQL pour la partie Back-End et HTML/CSS pour la partie Front-End.

Projet 7

Création d'un calendrier générique

Mise en place d'un calendrier générique qui permet de prendre des rendez-vous mais aussi d'en créer. Codé en SPring pour la partie Bck et en VueJs pour la partie Front.
